```css

  /* الأخضر الإسلامي */
  --primary: 160 60% 23%;                    /* #1a5f3f */
  --islamic-green-50: 156 100% 96%;
  --islamic-green-600: 160 60% 23%;
  --islamic-green-700: 160 65% 18%;

  /* الذهبي الإسلامي */
  --accent: 45 86% 58%;                      /* #d4af37 */
  --islamic-gold-400: 43 89% 56%;
  --islamic-gold-500: 38 92% 50%;

  /* التدرجات والظلال */
  --ring: 160 60% 23%;
}
```

### الوضع الليلي:
```css
.dark {
  --primary: 160 65% 18%;                    /* أخضر أغمق */
  --accent: 45 86% 58%;                      /* ذهبي ثابت */
}
```
