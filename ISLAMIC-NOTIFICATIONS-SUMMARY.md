## 🎯 المزايا الرئيسية

### 1️⃣ **المحتوى الإسلامي الغني**

#### الأذكار والتسبيح:
- 🌟 "سُبْحَانَ اللهِ وَبِحَمْدِهِ" - مائة حسنة
- ⭐ "لَا إِلَهَ إِلَّا اللهُ وَحْدَهُ لَا شَرِيكَ لَهُ" - خير من الدنيا وما فيها
- ❤️ "اللهُمَّ صَلِّ وَسَلِّمْ عَلَى نَبِيِّنَا مُحَمَّدٍ" - عشر صلوات من الله

#### الاستغفار والتوبة:
- 🤲 "أَسْتَغْفِرُ اللهَ الْعَظِيمَ الَّذِي لَا إِلَهَ إِلَّا هُوَ الْحَيُّ الْقَيُّومُ وَأَتُوبُ إِلَيْهِ"
- 🙏 "رَبِّ اغْفِرْ لِي ذَنْبِي وَخَطَئِي وَجَهْلِي"

#### الآيات القرآنية:
- 📖 "﴿ وَمَن يَتَّقِ اللَّهَ يَجْعَل لَّهُ مَخْرَجًا ﴾" - سورة الطلاق
- 📚 "﴿ وَمَن يَتَوَكَّلْ عَلَى اللَّهِ فَهُوَ حَسْبُهُ ﴾" - سورة الطلاق
- 📿 "﴿ فَاذْكُرُونِي أَذْكُرْكُمْ وَاشْكُرُوا لِي وَلَا تَكْفُرُونِ ﴾" - سورة البقرة
- 🌟 "﴿ إِنَّ مَعَ الْعُسْرِ يُسْرًا ﴾" - سورة الشرح

### 2️⃣ **التصميم الإسلامي المتميز**

#### ألوان متدرجة حسب المحتوى:
```jsx

  blue: { // للاستغفار
```

#### تأثيرات بصرية جميلة:
- ✨ **ظهور تدريجي** مع تأثيرات scale وfade
- 🌟 **خلفية ضبابية** (backdrop blur)
- 💫 **شريط متدرج علوي** حسب نوع المحتوى
- 🎭 **أيقونات مميزة** لكل نوع من المحتوى
- ✨ **تأثير إضاءة** خفيف في الخلفية

### 3️⃣ **الإعدادات القابلة للتخصيص**

#### التحكم الكامل:
```jsx
const [settings, setSettings] = useState({
  enabled: true,           // تفعيل/إيقاف الإشعارات
  interval: 300000,        // فترة العرض (5 دقائق افتراضياً)
  showDhikr: true,         // عرض الأذكار
  showIstighfar: true,     // عرض الاستغفار
  showVerses: true,        // عرض الآيات القرآنية
  autoHide: true,          // الإخفاء التلقائي
  autoHideDelay: 10000     // مدة الإخفاء التلقائي
});
```

#### خيارات الفترات الزمنية:
- ⏱️ **3 دقائق** للتذكير المتكرر
- ⏰ **5 دقائق** (افتراضي) للتوازن المثالي
- 🕐 **10 دقائق** للاستخدام المعتدل
- 🕒 **15 دقيقة** للاستخدام الخفيف
- 🕕 **30 دقيقة** للاستخدام النادر
- 🕘 **ساعة واحدة** للتذكير البسيط
