## 🎨 المعايير المطلوبة للهوية الإسلامية

### 1️⃣ **نظام الألوان الإسلامي**
```css
/* الألوان الأساسية المطلوبة */
--islamic-green-600: 160 60% 23%;    /* الأخضر الإسلامي */
--islamic-gold-500: 38 92% 50%;      /* الذهبي الإسلامي */
--islamic-green-50: 156 100% 96%;    /* أخضر فاتح للخلفيات */
--islamic-gold-50: 51 100% 96%;      /* ذهبي فاتح للخلفيات */
```

### 2️⃣ **خط Alexandria للنصوص العربية**
```css
.arabic-text {
  font-family: "Alexandria", "Inter", sans-serif !important;
}

.quran-verse {
  font-family: "Alexandria", serif !important;
  font-weight: 500;
  letter-spacing: 0.02em;
  line-height: 1.8;
}
```

### 3️⃣ **العناصر الإسلامية المطلوبة**
- ✅ **Header إسلامي** مع آية قرآنية
- ✅ **بطاقات إحصائيات** بألوان متدرجة إسلامية
- ✅ **أنماط هندسية** (islamic-pattern)
- ✅ **أيقونات ملونة** بتدرجات مناسبة
- ✅ **تأثيرات hover** ناعمة
- ✅ **دعم RTL** كامل

- 👥 **للمستخدمين**: "وَجَعَلْنَا مِنْهُمْ أَئِمَّةً يَهْدُونَ بِأَمْرِنَا"
- 📚 **للمشايخ**: "وَقُل رَّبِّ زِدْنِي عِلْماً"
- 📅 **للحجوزات**: "وَاصْبِرْ نَفْسَكَ مَعَ الَّذِينَ يَدْعُونَ رَبَّهُم"
